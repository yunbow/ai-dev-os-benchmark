# AI Dev OS Benchmark への貢献

貢献に興味を持っていただきありがとうございます！

## 貢献方法

### Issue の報告
- [GitHub Issues](https://github.com/yunbow/ai-dev-os-benchmark/issues) でエラーの報告や改善の提案ができます
- 評価項目に関する報告にはファイルパスとチェック項目 ID（例: S6, E1）を含めてください

### Pull Request
1. リポジトリをフォーク
2. フィーチャーブランチを作成（`git checkout -b improve-evaluation`）
3. 変更を加える
4. 説明的なコミットメッセージでコミット
5. フォークに push して Pull Request を作成

### 貢献できる内容

| ディレクトリ | 求めている内容 | ガイドライン |
|-----------|-------------|------------|
| `spec/` | ベンチマーク設計の改善、新しい評価軸 | ベンチマーク全体に影響。まず Issue で議論。 |
| `assets/prompts/` | より良いプロンプトテンプレート、追加条件 | 再現性を維持すること。 |
| `assets/` | 評価シートの改善、新しいチェック項目 | 客観的・機械的に評価可能な項目にすること。 |
| `results/` | ベンチマーク実行結果、分析の改善 | メタデータ（モデル、日時、commit hash）を含めること。 |
| `docs/i18n/` | 翻訳、翻訳の改善 | 下記の翻訳ガイドを参照。 |

### 翻訳ガイド

- 英語ファイルが原本
- 翻訳は `docs/i18n/{lang}/` に英語と同じ構造で配置
- 現在対応: `ja`（日本語）
- 技術用語、コードブロック、URL、ファイルパスは英語のまま
- 各ファイル末尾に `Languages:` フッターで英語版へのリンクを追加
