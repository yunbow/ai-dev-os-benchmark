# テスト 003: YAML チェックリスト形式 — 構造化メタデータは AI の遵守率を改善するか？

## 結果

**改善なし。** security.md に YAML frontmatter + Quick Rules サマリー（+165行、+33%）を追加しても遵守率は改善しなかった。

| タスク | A（散文のみ） | C（YAML+Quick Rules） | 勝者 |
|--------|:---:|:---:|:---:|
| 1. ログイン API | 5/5 | 5/5 | 引き分け |
| 2. タスク CRUD | 4/4 | 3/4 | **A** |
| 3. チーム招待 | 3/3 | 3/3 | 引き分け |
| 4. Webhook ハンドラ | 3/3 | 3/3 | 引き分け |
| 5. ミドルウェアヘッダー | 3/3 | 3/3 | 引き分け |
| **合計** | **18/18** | **17/18** | **A** |

散文のみの条件が高スコア。YAML と Quick Rules は散文本文の情報を重複させるだけ — 情報を追加せずトークンを追加。

**判定:** ガイドラインに YAML frontmatter や Quick Rules を追加しない。散文の明確さに注力。
